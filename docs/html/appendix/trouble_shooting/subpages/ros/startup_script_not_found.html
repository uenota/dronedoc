もしかして¶

~/.ros/etc/init.d-posix 以下にrcSというファイルがありますか？

PX4 SITLシミュレーションでは最初にスタートアップスクリプト（ ~/.ros/etc/init.d-posix ）がロードされます。 普通はセットアップの際に自動で ~/src/Firmware/ROMFS/px4fmu_common から ~/.ros/etc へのシンボリックリンクが作成されますが、何らかの理由で作成されなかったのだと考えられます。

以下のコマンドを使って``~/src/Firmware/ROMFS/px4fmu_common`` から ~/.ros/etc へのシンボリックリンクを作成しましょう。

ln -s ~/src/Firmware/ROMFS/px4fmu_common ~/.ros/etc
